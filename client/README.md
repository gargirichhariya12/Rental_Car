# Client App

This folder contains the React + Vite frontend for the Rental Car app.

For full project documentation, setup, environment variables, routes, and backend details, see the repo root README:

`../README.md`

## Client Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Required Client Env

Create `client/.env`:

```env
VITE_BASE_URL=http://localhost:3000
VITE_CURRENCY=$
```
